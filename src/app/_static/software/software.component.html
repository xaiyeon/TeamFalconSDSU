Software

At Team Falcon we decided to go with the Teensy 3.2 board because of the (MK20DX256) 32-bit ARM Cortex-M4 @72 MHz and additional prototyping on a PIC32MX6410B. Respectively, the integrated development environment (IDE) used were Arduino Sketch and MPLAB by MicroChip. Both IDEs used the C programming language.

The Code

We use a few libraries for faster prototyping. This gives us more time to program strategies and hasten development process.


How the sensors work:

First is the IR Receiver module, which will start our robot.


Next are our edge sensors that will determine black or white ground.


Then finally, we have the SHARP IR sensors for finding our enemy.
